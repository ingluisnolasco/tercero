Consejos, ejercicios tipo examen y soluciones ocultas

1. Problema de tasa constante (tanque)

Razón
Proporcionalidad
Sustracción
Pasos para resolver
  • Identifica la cantidad inicial (capacidad o nivel actual) y la tasa de cambio por unidad de tiempo
  • Multiplica la tasa por el tiempo transcurrido y réstalo de la cantidad inicial
  • Comprueba que el resultado no sea negativo y que respete límites físicos (0 a capacidad máxima)
Ejemplo tipo examen: "Capacidad 400 L; se vacía a 0.8 L/min. ¿Cuántos litros quedan después de 10 min?"

Ejemplo tipo examen: \(7f - 12 + f = -12 + 4f + 20\)

3. Ecuaciones de segundo grado (tres formas)

\(ax^2 + c = 0\)
\(ax^2 + bx = 0\)
\(ax^2 + bx + c = 0\)
Cómo abordarlas
  • \(ax^2 + c = 0\): despejar \(x^2\) y usar raíz cuadrada; recuerda valores absolutos si corresponde
  • \(ax^2 + bx = 0\): factoriza \(x(ax + b)=0\) y aplica la propiedad del producto nulo
  • \(ax^2 + bx + c = 0\): aplica la fórmula general; en el examen la fórmula estará disponible
Ejemplos tipo examen: \(2x^2 - 8 = 0\); \(x^2 - 5x = 0\); \(x^2 - 3x + 2 = 0\)

Problema de tanque — práctica

✕

Resuelve y comprueba

  1. Capacidad 400 L; se vacía a 0.8 L/min. ¿Cuántos litros quedan después de 10 min?
  2. Si en lugar de vaciarse se llenara a 0.8 L/min, ¿cuánto tiempo tardaría en llenarse desde 0 hasta 400 L?
  3. Si el tanque tenía 250 L y se vacía a 1.2 L/min, ¿cuánto queda tras 60 min?
  1. \(400 - 0.8\times10 = 400 - 8 = 392\) litros.
  2. Tiempo = \(400 \div 0.8 = 500\) minutos.
  3. \(250 - 1.2\times60 = 250 - 72 = 178\) litros.

Ecuaciones de primer grado — práctica

✕

Resuelve cada una; la variable puede ser cualquier letra

  1. \(7f - 12 + f = -12 + 4f + 20\)
  2. \(3a + 5 = 2a - 7\)
  3. \(5y - 2( y - 3) = 4\)
  4. \(2m + 3 = 3( m - 1) + 6\)
  1. Simplifica: \(8f -12 = 4f +8\) → \(4f = 20\) → \(f = 5\).
  2. \(3a+5=2a-7\) → \(a=-12\).
  3. \(5y -2y +6 =4\) → \(3y = -2\) → \(y = -\tfrac{2}{3}\).
  4. \(2m+3 = 3m -3 +6\) → \(2m+3 = 3m+3\) → \(-m=0\) → \(m=0\).

Ecuaciones de segundo grado — práctica

✕

Aplica el método indicado para cada forma

  1. \(2x^2 - 8 = 0\) (forma \(ax^2 + c = 0\))
  2. \(x^2 - 5x = 0\) (forma \(ax^2 + bx = 0\))
  3. \(x^2 - 3x + 2 = 0\) (forma general)
  1. \(2x^2 = 8\) → \(x^2 = 4\) → \(x = \pm 2\). (usar valor absoluto si se presenta como \(x = \sqrt{4}\))
  2. Factoriza: \(x(x - 5) = 0\) → \(x = 0\) o \(x = 5\).
  3. Fórmula general: raíces \(x = \dfrac{3 \pm \sqrt{9 - 8}}{2} = \dfrac{3 \pm 1}{2}\) → \(x=2\) o \(x=1\).

Sistemas de ecuaciones — práctica

✕

Resuelve por el método que prefieras

  1. \(\begin{cases}2x + 3y = 7\\ x - y = 1\end{cases}\)
  2. \(\begin{cases}3a - 2b = 4\\ 6a - 4b = 8\end{cases}\) (observa la relación entre ecuaciones)
  3. \(\begin{cases}x + 2y = 5\\ 2x - y = 1\end{cases}\)
  1. De \(x - y =1\) → \(x = y+1\). Sustituye: \(2(y+1)+3y=7\) → \(5y+2=7\) → \(y=1\), \(x=2\).
  2. Segunda ecuación es múltiplo de la primera → infinitas soluciones si coinciden; aquí ambas son equivalentes → infinitas soluciones (coincidentes).
  3. Multiplica segunda por 2 y resta: \(x+2y=5\) y \(4x-2y=2\) → suma: \(5x=7\) → \(x=7/5\); luego \(y=(5-x)/2\).

Inecuaciones y problema aplicado — práctica

✕

Grafica y modela; da ejemplos válidos y no válidos

  1. Grafica la inecuación \(2x + y \le 6\)
  2. Grafica la inecuación \(x - 3y > 2\)
  3. Problema: cooperativa — dorado = \(d\), camarón = \(c\). Condiciones: \(d + c \le 800\) y \(c \le 2d\). Da 3 combinaciones válidas y 3 no válidas y explica.
  1. Recta \(2x+y=6\). Sombrea la región por debajo (incluye la recta). Usa línea continua.
  2. Recta \(x-3y=2\). Sombrea el lado donde \(x-3y>2\). Usa línea punteada.
  3. Ejemplos válidos: (d=200,c=400) cumple \(200+400=600\le800\) y \(400\le400\). (d=300,c=600) cumple. (d=400,c=0) cumple. No válidos: (d=100,c=500) porque \(500>2\cdot100\). (d=500,c=400) porque \(500+400>800\). (d=0,c=700) porque \(700>2\cdot0\).

Polinomios — práctica (operaciones y Ruffini)

✕

Realiza operaciones y división

  1. Dados \(A(x)=2x+3\) y \(B(x)=x^2\): calcula \(A+B\), \(A-B\), \(A\cdot B\), \(A\div B\) (si aplica)
  2. Divide \(x^3 - 6x^2 + 11x -6\) por \(x - 1\) usando Ruffini
  3. Clasifica: ¿qué es un monomio, binomio, trinomio? Da ejemplos
  1. \(A+B = x^2 + 2x + 3\). \(A-B = -x^2 + 2x + 3\). \(A\cdot B = 2x^3 + 3x^2\). División \(A\div B\) no es polinomio entero (grado menor).
  2. Ruffini con raíz \(1\): coeficientes 1, -6, 11, -6 → cociente \(x^2 -5x +6\) y resto 0.
  3. Monomio: una sola parte (ej. \(5x^3\)). Binomio: dos términos (ej. \(x+2\)). Trinomio: tres términos (ej. \(x^2 -3x +2\)).

Diagramas de Venn — práctica

✕

Dibuja y completa

  1. Conjunto A tiene 12 elementos, B tiene 9, y la intersección A∩B tiene 4. Dibuja el diagrama y coloca los números en cada región.
  2. Si universo tiene 25 elementos, ¿cuántos están fuera de A∪B?
  3. Dibuja un diagrama de Euler si A∩B = ∅ y A tiene 7 elementos, B tiene 5.
  1. Región A sólo: \(12-4=8\). Región B sólo: \(9-4=5\). Intersección: 4.
  2. \(|A\cup B| = 8+5+4 = 17\). Fuera: \(25-17=8\).
  3. Euler: dibuja dos círculos separados (sin intersección) y coloca 7 y 5 respectivamente; resto del universo según datos.
